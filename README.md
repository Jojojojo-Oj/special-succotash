# AGAPAY APP

A mobile emergency response application built using Flutter that helps users quickly send SOS alerts, share real-time location, and notify designated contacts or organizations during emergencies.

---

##  Features

-  SOS Emergency Button for instant alerts  
-  Real-time location tracking of users in distress  
-  SMS notification system for emergency contacts  
-  User authentication system  
-  Cloud database integration for storing user and emergency data  
-  Faster emergency reporting for response coordination  
-  Weather API
-  Can Send SOS signal without internet and signal (VIA BLE)
---

##  Tech Stack

- **Flutter** – Frontend mobile development  
- **Dart** – Programming language  
- **Firebase** – Backend services (Auth, Firestore, etc.)  
- **Google Maps / Location Services** – For geolocation tracking  
- **SMS Service** – For emergency alerts
- **OpenWeather** - for weather updates
